## Program Goal

### One Sentence

Demonstrate supervised autonomy working on a single rover (SLAM + waypoint following + obstacle avoidance), with a compelling live demo that shows the path to 1:10 operator ratio.

### Why This Goal

- Business model depends on supervised autonomy (10x cost difference)
- Investors need to see capability, not just hear promises
- Indoor demo proves the autonomy stack (snow is just an attachment)
- Maps directly to Q3 2026 milestone in business plan

### Success Criteria

- [ ] Rover autonomously patrols a course for 10+ consecutive runs
- [ ] Obstacle avoidance works with unpredictable humans
- [ ] Operator handoff is smooth and reliable
- [ ] At least 3 investor meetings scheduled from demo day
- [ ] LOI or strong interest from 1+ investor

## Demo Day Vision

### Core Insight

Investors remember **experiences**, not pitches. Give them something to do, not just watch.

### The Demo: "Step Into The Course"

**Setup:** Rover autonomously patrols a marked course. A sign invites people to walk into its path.

**The Moment:**

1. Investor steps in front of rover
2. Rover stops immediately
3. Rover recalculates path
4. Rover navigates around them
5. Rover continues on its way

**Why It Works:**

- Interactive (they're part of it)
- Proves the tech (not a video, not a simulation)
- Creates a story ("I stood in front of a robot and it figured out how to go around me")
- Shareable ("You have to go see the sidewalk robot")

### Level Up: Let Them Drive

After the autonomous demo:

> "Want to try? Just push forward."

Hand them the Xbox controller. They drive for 10 seconds. Then:

> "Now imagine ten of these. One operator. That's the business."

**Why It Works:**

- Tactile memory (they felt the controls)
- Power fantasy (they controlled a real robot)
- Immediate understanding of teleop â†’ autonomy transition

### Differential Drive Odometry

VESC reports wheel velocities. Proper odometry integrates these with covariance:

```rust
/// Differential drive odometry with uncertainty propagation
pub struct DifferentialOdometry {
    /// Current pose in odom frame
    pose: Isometry2<f64>,
    /// Pose covariance (3x3: x, y, theta)
    covariance: Matrix3<f64>,
    /// Physical parameters
    wheel_radius: f64,
    track_width: f64,
    /// Noise parameters (tune empirically)
    alpha: [f64; 4],  // Motion model noise
}

impl DifferentialOdometry {
    pub fn update(&mut self, left_vel: f64, right_vel: f64, dt: f64) -> Isometry2<f64> {
        // Compute body velocities
        let v = (left_vel + right_vel) / 2.0 * self.wheel_radius;
        let omega = (right_vel - left_vel) / self.track_width * self.wheel_radius;

        // Integrate pose (exact integration for differential drive)
        let delta = if omega.abs() < 1e-6 {
            // Straight line motion
            Vector3::new(v * dt, 0.0, 0.0)
        } else {
            // Arc motion
            let r = v / omega;
            Vector3::new(
                r * omega.sin() * dt,
                r * (1.0 - (omega * dt).cos()),
                omega * dt,
            )
        };

        // Transform delta to world frame and apply
        let delta_world = self.pose.rotation * delta.xy();
        self.pose = Isometry2::new(
            self.pose.translation.vector + delta_world,
            self.pose.rotation.angle() + delta.z,
        );

        // Propagate covariance using motion model Jacobian
        self.propagate_covariance(v, omega, dt);

        self.pose
    }

    fn propagate_covariance(&mut self, v: f64, omega: f64, dt: f64) {
        let theta = self.pose.rotation.angle();

        // Jacobian of motion model w.r.t. state
        let g = Matrix3::new(
            1.0, 0.0, -v * dt * theta.sin(),
            0.0, 1.0,  v * dt * theta.cos(),
            0.0, 0.0,  1.0,
        );

        // Motion noise (velocity-dependent)
        let v_var = self.alpha[0] * v.abs() + self.alpha[1] * omega.abs();
        let omega_var = self.alpha[2] * v.abs() + self.alpha[3] * omega.abs();
        let q = Matrix3::new(
            v_var * dt, 0.0, 0.0,
            0.0, v_var * dt, 0.0,
            0.0, 0.0, omega_var * dt,
        );

        self.covariance = g * self.covariance * g.transpose() + q;
    }
}
```

### Extended Kalman Filter

Fuse odometry with scan matching corrections:

```rust
/// EKF for pose estimation
/// State: [x, y, theta]
pub struct PoseEkf {
    /// Current state estimate
    state: Vector3<f64>,
    /// State covariance
    covariance: Matrix3<f64>,
}

impl PoseEkf {
    /// Prediction step (from odometry)
    pub fn predict(&mut self, odom_delta: &Isometry2<f64>, odom_cov: &Matrix3<f64>) {
        let theta = self.state.z;

        // Apply motion model
        let cos_t = theta.cos();
        let sin_t = theta.sin();
        let dx = odom_delta.translation.x;
        let dy = odom_delta.translation.y;
        let dtheta = odom_delta.rotation.angle();

        self.state += Vector3::new(
            cos_t * dx - sin_t * dy,
            sin_t * dx + cos_t * dy,
            dtheta,
        );
        self.state.z = normalize_angle(self.state.z);

        // Jacobian
        let g = Matrix3::new(
            1.0, 0.0, -sin_t * dx - cos_t * dy,
            0.0, 1.0,  cos_t * dx - sin_t * dy,
            0.0, 0.0,  1.0,
        );

        self.covariance = g * self.covariance * g.transpose() + odom_cov;
    }

    /// Update step (from scan matching)
    pub fn update(&mut self, measured_pose: &Vector3<f64>, measurement_cov: &Matrix3<f64>) {
        // Innovation
        let y = measured_pose - self.state;
        let y = Vector3::new(y.x, y.y, normalize_angle(y.z));

        // Kalman gain (H = I for direct pose measurement)
        let s = self.covariance + measurement_cov;
        let k = self.covariance * s.try_inverse().unwrap_or(Matrix3::identity());

        // Update state and covariance
        self.state += k * y;
        self.state.z = normalize_angle(self.state.z);
        self.covariance = (Matrix3::identity() - k) * self.covariance;
    }

    pub fn pose(&self) -> Isometry2<f64> {
        Isometry2::new(Vector2::new(self.state.x, self.state.y), self.state.z)
    }

    pub fn covariance(&self) -> &Matrix3<f64> {
        &self.covariance
    }
}

fn normalize_angle(angle: f64) -> f64 {
    let mut a = angle % (2.0 * std::f64::consts::PI);
    if a > std::f64::consts::PI {
        a -= 2.0 * std::f64::consts::PI;
    } else if a < -std::f64::consts::PI {
        a += 2.0 * std::f64::consts::PI;
    }
    a
}
```

### Scan Matching: Correlative Scan Matcher

More robust than ICP for 2D LiDAR. Handles larger initial errors:

```rust
/// Correlative scan matcher
/// Searches a grid of possible poses to find the best alignment
pub struct CorrelativeScanMatcher {
    /// Search window in x, y, theta
    linear_search_window: f64,   // meters
    angular_search_window: f64,  // radians
    /// Resolution
    linear_resolution: f64,      // meters
    angular_resolution: f64,     // radians
    /// Lookup table for fast matching
    lookup_table: Option<LookupTable>,
}

impl CorrelativeScanMatcher {
    /// Match scan to reference (map or previous scan)
    pub fn match_scan(
        &self,
        scan: &Scan,
        reference: &OccupancyGrid,
        initial_pose: Isometry2<f64>,
    ) -> ScanMatchResult {
        // Build precomputed lookup table if needed
        let lookup = self.build_lookup_table(reference);

        // Coarse search: find approximate best pose
        let coarse_result = self.coarse_search(scan, &lookup, initial_pose);

        // Fine search: refine with gradient descent
        let fine_result = self.fine_search(scan, reference, coarse_result.pose);

        ScanMatchResult {
            pose: fine_result.pose,
            score: fine_result.score,
            covariance: self.estimate_covariance(scan, reference, fine_result.pose),
        }
    }

    fn coarse_search(
        &self,
        scan: &Scan,
        lookup: &LookupTable,
        initial: Isometry2<f64>,
    ) -> SearchResult {
        let mut best_score = f64::NEG_INFINITY;
        let mut best_pose = initial;

        // Grid search over pose space
        let x_steps = (2.0 * self.linear_search_window / self.linear_resolution) as i32;
        let y_steps = x_steps;
        let theta_steps = (2.0 * self.angular_search_window / self.angular_resolution) as i32;

        for xi in -x_steps..=x_steps {
            for yi in -y_steps..=y_steps {
                for ti in -theta_steps..=theta_steps {
                    let dx = xi as f64 * self.linear_resolution;
                    let dy = yi as f64 * self.linear_resolution;
                    let dtheta = ti as f64 * self.angular_resolution;

                    let test_pose = Isometry2::new(
                        initial.translation.vector + Vector2::new(dx, dy),
                        initial.rotation.angle() + dtheta,
                    );

                    let score = self.score_pose(scan, lookup, test_pose);
                    if score > best_score {
                        best_score = score;
                        best_pose = test_pose;
                    }
                }
            }
        }

        SearchResult { pose: best_pose, score: best_score }
    }

    fn score_pose(&self, scan: &Scan, lookup: &LookupTable, pose: Isometry2<f64>) -> f64 {
        let mut score = 0.0;

        for point in scan.points_in_frame(&pose) {
            score += lookup.get_probability(point.x, point.y);
        }

        score / scan.points.len() as f64
    }

    /// Estimate covariance from the score surface curvature
    fn estimate_covariance(
        &self,
        scan: &Scan,
        reference: &OccupancyGrid,
        pose: Isometry2<f64>,
    ) -> Matrix3<f64> {
        // Compute Hessian of score function at optimum
        let h = 1e-4;
        let score_center = self.score_at(scan, reference, pose);

        let mut hessian = Matrix3::zeros();

        // Numerical second derivatives
        for i in 0..3 {
            for j in i..3 {
                let pp = self.perturb_pose(pose, i, h, j, h);
                let pm = self.perturb_pose(pose, i, h, j, -h);
                let mp = self.perturb_pose(pose, i, -h, j, h);
                let mm = self.perturb_pose(pose, i, -h, j, -h);

                let d2 = (self.score_at(scan, reference, pp)
                        - self.score_at(scan, reference, pm)
                        - self.score_at(scan, reference, mp)
                        + self.score_at(scan, reference, mm)) / (4.0 * h * h);

                hessian[(i, j)] = -d2;  // Negative because we want covariance, not precision
                hessian[(j, i)] = -d2;
            }
        }

        // Invert Hessian to get covariance (add regularization for stability)
        (hessian + Matrix3::identity() * 1e-6)
            .try_inverse()
            .unwrap_or(Matrix3::identity() * 0.1)
    }
}

pub struct ScanMatchResult {
    pub pose: Isometry2<f64>,
    pub score: f64,
    pub covariance: Matrix3<f64>,
}
```

### Pose Graph SLAM

For loop closure and global consistency:

```rust
/// Pose graph node
#[derive(Clone)]
pub struct PoseNode {
    pub id: usize,
    pub pose: Isometry2<f64>,
    pub scan: Arc<Scan>,  // Reference to scan at this pose
    pub timestamp: Instant,
}

/// Pose graph edge (constraint)
#[derive(Clone)]
pub struct PoseEdge {
    pub from: usize,
    pub to: usize,
    pub measurement: Isometry2<f64>,
    pub information: Matrix3<f64>,  // Inverse covariance
}

/// Pose graph SLAM
pub struct PoseGraph {
    nodes: Vec<PoseNode>,
    edges: Vec<PoseEdge>,
    scan_matcher: CorrelativeScanMatcher,
    /// Minimum distance between nodes
    node_spacing: f64,
    /// Loop closure detection threshold
    loop_closure_distance: f64,
}

impl PoseGraph {
    pub fn add_scan(&mut self, scan: Scan, odom_pose: Isometry2<f64>) -> Option<usize> {
        // Check if we should add a new node
        if let Some(last) = self.nodes.last() {
            let dist = (odom_pose.translation.vector - last.pose.translation.vector).norm();
            if dist < self.node_spacing {
                return None;
            }
        }

        let node_id = self.nodes.len();

        // Add odometry edge from previous node
        if let Some(prev) = self.nodes.last() {
            let delta = prev.pose.inverse() * odom_pose;
            self.edges.push(PoseEdge {
                from: prev.id,
                to: node_id,
                measurement: delta,
                information: Matrix3::identity() * 100.0,  // Tune based on odometry quality
            });
        }

        // Check for loop closures
        self.detect_loop_closures(node_id, &scan, odom_pose);

        // Add node
        self.nodes.push(PoseNode {
            id: node_id,
            pose: odom_pose,
            scan: Arc::new(scan),
            timestamp: Instant::now(),
        });

        // Optimize if we added a loop closure
        if self.has_recent_loop_closure() {
            self.optimize();
        }

        Some(node_id)
    }

    fn detect_loop_closures(&mut self, current_id: usize, scan: &Scan, pose: Isometry2<f64>) {
        for node in &self.nodes {
            // Skip recent nodes (they're connected by odometry)
            if current_id.saturating_sub(node.id) < 10 {
                continue;
            }

            // Check if close enough to be a potential loop closure
            let dist = (pose.translation.vector - node.pose.translation.vector).norm();
            if dist > self.loop_closure_distance {
                continue;
            }

            // Try to match scans
            let result = self.scan_matcher.match_scan_to_scan(scan, &node.scan, pose);

            if result.score > 0.8 {  // Good match
                let delta = node.pose.inverse() * result.pose;
                self.edges.push(PoseEdge {
                    from: node.id,
                    to: current_id,
                    measurement: delta,
                    information: result.covariance.try_inverse()
                        .unwrap_or(Matrix3::identity() * 10.0),
                });
            }
        }
    }

    /// Optimize pose graph using Levenberg-Marquardt
    pub fn optimize(&mut self) {
        const MAX_ITERATIONS: usize = 100;
        const TOLERANCE: f64 = 1e-6;

        let mut lambda = 1e-3;  // LM damping factor

        for _ in 0..MAX_ITERATIONS {
            // Build linear system: H * dx = b
            let (h, b) = self.build_linear_system();

            // Solve with LM damping
            let h_damped = h + Matrix::<f64, Dyn, Dyn, _>::identity(h.nrows(), h.ncols()) * lambda;

            let dx = match h_damped.lu().solve(&b) {
                Some(x) => x,
                None => break,  // Singular matrix
            };

            // Check convergence
            if dx.norm() < TOLERANCE {
                break;
            }

            // Apply update
            self.apply_update(&dx);

            // Adjust lambda based on error reduction
            // (simplified: just decrease)
            lambda *= 0.5;
        }
    }

    fn build_linear_system(&self) -> (DMatrix<f64>, DVector<f64>) {
        let n = self.nodes.len() * 3;  // 3 DOF per node
        let mut h = DMatrix::zeros(n, n);
        let mut b = DVector::zeros(n);

        for edge in &self.edges {
            let i = edge.from * 3;
            let j = edge.to * 3;

            // Compute error and Jacobian
            let (error, jacobian_i, jacobian_j) = self.compute_edge_error(edge);

            // Add to H and b
            // H += J^T * Omega * J
            // b += J^T * Omega * e
            let omega = &edge.information;

            h.slice_mut((i, i), (3, 3)).add_assign(jacobian_i.transpose() * omega * &jacobian_i);
            h.slice_mut((i, j), (3, 3)).add_assign(jacobian_i.transpose() * omega * &jacobian_j);
            h.slice_mut((j, i), (3, 3)).add_assign(jacobian_j.transpose() * omega * &jacobian_i);
            h.slice_mut((j, j), (3, 3)).add_assign(jacobian_j.transpose() * omega * &jacobian_j);

            b.rows_mut(i, 3).add_assign(jacobian_i.transpose() * omega * &error);
            b.rows_mut(j, 3).add_assign(jacobian_j.transpose() * omega * &error);
        }

        // Fix first node (gauge freedom)
        for k in 0..3 {
            h[(k, k)] += 1e10;
        }

        (h, b)
    }
}
```

## Occupancy Grid Map

### Log-Odds Representation

Proper Bayesian map update with log-odds for numerical stability:

```rust
/// Occupancy grid with log-odds representation
pub struct OccupancyGrid {
    /// Grid data in log-odds: positive = occupied, negative = free
    data: Array2<f32>,
    /// Grid resolution (meters per cell)
    resolution: f64,
    /// Origin of grid in world frame (bottom-left corner)
    origin: Vector2<f64>,
    /// Update parameters
    log_odds_hit: f32,
    log_odds_miss: f32,
    log_odds_max: f32,
    log_odds_min: f32,
}

impl OccupancyGrid {
    pub fn new(width: usize, height: usize, resolution: f64, origin: Vector2<f64>) -> Self {
        Self {
            data: Array2::zeros((height, width)),
            resolution,
            origin,
            log_odds_hit: 0.85_f32.ln() - 0.15_f32.ln(),   // p=0.85
            log_odds_miss: 0.4_f32.ln() - 0.6_f32.ln(),    // p=0.4
            log_odds_max: 0.97_f32.ln() - 0.03_f32.ln(),   // Clamp to avoid certainty
            log_odds_min: 0.03_f32.ln() - 0.97_f32.ln(),
        }
    }

    /// Integrate a scan into the map using ray casting
    pub fn integrate_scan(&mut self, scan: &Scan, pose: &Isometry2<f64>) {
        let sensor_pos = pose.translation.vector;

        for (range, angle) in scan.iter() {
            if !range.is_finite() || *range < scan.min_range || *range > scan.max_range {
                continue;
            }

            // Compute endpoint in world frame
            let world_angle = pose.rotation.angle() + angle;
            let endpoint = sensor_pos + Vector2::new(
                range * world_angle.cos(),
                range * world_angle.sin(),
            );

            // Ray cast from sensor to endpoint
            self.trace_ray(sensor_pos, endpoint, *range < scan.max_range - 0.1);
        }
    }

    fn trace_ray(&mut self, start: Vector2<f64>, end: Vector2<f64>, hit_obstacle: bool) {
        // Bresenham's line algorithm
        let start_cell = self.world_to_grid(start);
        let end_cell = self.world_to_grid(end);

        let mut x = start_cell.0 as i32;
        let mut y = start_cell.1 as i32;
        let x1 = end_cell.0 as i32;
        let y1 = end_cell.1 as i32;

        let dx = (x1 - x).abs();
        let dy = (y1 - y).abs();
        let sx = if x < x1 { 1 } else { -1 };
        let sy = if y < y1 { 1 } else { -1 };
        let mut err = dx - dy;

        loop {
            // Mark cell as free (all cells along ray except last)
            if x != x1 || y != y1 {
                self.update_cell(x as usize, y as usize, false);
            }

            if x == x1 && y == y1 {
                break;
            }

            let e2 = 2 * err;
            if e2 > -dy {
                err -= dy;
                x += sx;
            }
            if e2 < dx {
                err += dx;
                y += sy;
            }
        }

        // Mark endpoint as occupied (if we hit something)
        if hit_obstacle {
            self.update_cell(end_cell.0, end_cell.1, true);
        }
    }

    fn update_cell(&mut self, x: usize, y: usize, occupied: bool) {
        if x >= self.data.ncols() || y >= self.data.nrows() {
            return;
        }

        let log_odds_update = if occupied { self.log_odds_hit } else { self.log_odds_miss };
        let current = self.data[(y, x)];
        let updated = (current + log_odds_update).clamp(self.log_odds_min, self.log_odds_max);
        self.data[(y, x)] = updated;
    }

    /// Get probability of occupancy (0.0 = free, 1.0 = occupied)
    pub fn probability(&self, x: usize, y: usize) -> f64 {
        if x >= self.data.ncols() || y >= self.data.nrows() {
            return 0.5;  // Unknown
        }

        let log_odds = self.data[(y, x)] as f64;
        1.0 / (1.0 + (-log_odds).exp())
    }

    /// Convert world coordinates to grid cell
    pub fn world_to_grid(&self, point: Vector2<f64>) -> (usize, usize) {
        let local = point - self.origin;
        let x = (local.x / self.resolution).floor() as usize;
        let y = (local.y / self.resolution).floor() as usize;
        (x.min(self.data.ncols() - 1), y.min(self.data.nrows() - 1))
    }

    /// Convert grid cell to world coordinates (cell center)
    pub fn grid_to_world(&self, x: usize, y: usize) -> Vector2<f64> {
        self.origin + Vector2::new(
            (x as f64 + 0.5) * self.resolution,
            (y as f64 + 0.5) * self.resolution,
        )
    }
}
```

### Hybrid A\* for Global Planning

Produces kinematically feasible paths (unlike grid A\*):

```rust
/// Hybrid A* state: continuous pose discretized for search
#[derive(Clone, Copy, PartialEq, Eq, Hash)]
pub struct HybridState {
    x: i32,      // Discretized x
    y: i32,      // Discretized y
    theta: i32,  // Discretized heading (e.g., 72 bins = 5Â° resolution)
}

/// Motion primitives for differential drive
pub struct MotionPrimitive {
    pub linear_vel: f64,
    pub angular_vel: f64,
    pub duration: f64,
}

pub struct HybridAStar {
    /// Grid resolution for state discretization
    xy_resolution: f64,
    theta_resolution: f64,
    /// Motion primitives to try at each expansion
    primitives: Vec<MotionPrimitive>,
    /// Heuristic: 2D A* ignoring heading (admissible)
    heuristic_map: Option<Array2<f64>>,
}

impl HybridAStar {
    pub fn plan(
        &mut self,
        start: Isometry2<f64>,
        goal: Isometry2<f64>,
        costmap: &Costmap,
    ) -> Option<Vec<Isometry2<f64>>> {
        // Precompute heuristic (2D Dijkstra from goal)
        self.compute_heuristic(goal, costmap);

        let start_state = self.discretize(start);
        let goal_state = self.discretize(goal);

        // Priority queue: (f-score, state)
        let mut open = BinaryHeap::new();
        let mut came_from: HashMap<HybridState, (HybridState, Isometry2<f64>)> = HashMap::new();
        let mut g_score: HashMap<HybridState, f64> = HashMap::new();

        g_score.insert(start_state, 0.0);
        open.push(Reverse((OrderedFloat(self.heuristic(start)), start_state, start)));

        while let Some(Reverse((_, current_state, current_pose))) = open.pop() {
            // Check if we reached the goal
            if self.is_goal(current_state, goal_state, current_pose, goal) {
                return Some(self.reconstruct_path(came_from, current_state, goal));
            }

            let current_g = g_score[&current_state];

            // Expand using motion primitives
            for primitive in &self.primitives {
                let (next_pose, cost) = self.apply_primitive(current_pose, primitive, costmap);

                if cost.is_infinite() {
                    continue;  // Collision
                }

                let next_state = self.discretize(next_pose);
                let tentative_g = current_g + cost;

                if tentative_g < *g_score.get(&next_state).unwrap_or(&f64::INFINITY) {
                    came_from.insert(next_state, (current_state, next_pose));
                    g_score.insert(next_state, tentative_g);

                    let f = tentative_g + self.heuristic(next_pose);
                    open.push(Reverse((OrderedFloat(f), next_state, next_pose)));
                }
            }
        }

        None  // No path found
    }

    fn apply_primitive(
        &self,
        pose: Isometry2<f64>,
        primitive: &MotionPrimitive,
        costmap: &Costmap,
    ) -> (Isometry2<f64>, f64) {
        // Simulate motion
        let dt = 0.1;
        let steps = (primitive.duration / dt) as usize;

        let mut current = pose;
        let mut total_cost = 0.0;

        for _ in 0..steps {
            // Differential drive kinematics
            let theta = current.rotation.angle();
            let dx = primitive.linear_vel * dt * theta.cos();
            let dy = primitive.linear_vel * dt * theta.sin();
            let dtheta = primitive.angular_vel * dt;

            current = Isometry2::new(
                current.translation.vector + Vector2::new(dx, dy),
                theta + dtheta,
            );

            // Check collision and add cost
            let cell_cost = costmap.cost_at(current.translation.vector);
            if cell_cost >= LETHAL_COST {
                return (current, f64::INFINITY);
            }
            total_cost += cell_cost + primitive.duration / steps as f64;
        }

        (current, total_cost)
    }
}
```

### Dynamic Window Approach for Local Planning

Reactive obstacle avoidance with smooth motion:

```rust
/// Dynamic Window Approach local planner
pub struct DwaPlanner {
    // Velocity limits
    max_linear_vel: f64,
    min_linear_vel: f64,
    max_angular_vel: f64,
    max_linear_acc: f64,
    max_angular_acc: f64,

    // Simulation parameters
    sim_time: f64,
    sim_granularity: f64,

    // Cost weights
    path_distance_weight: f64,
    goal_distance_weight: f64,
    obstacle_weight: f64,

    // Velocity sampling
    linear_vel_samples: usize,
    angular_vel_samples: usize,
}

#[derive(Clone)]
pub struct Trajectory {
    pub poses: Vec<Isometry2<f64>>,
    pub velocities: Vec<Twist>,
    pub time_step: f64,
}

impl DwaPlanner {
    pub fn compute_velocity(
        &self,
        current_pose: Isometry2<f64>,
        current_vel: Twist,
        global_path: &[Isometry2<f64>],
        costmap: &Costmap,
    ) -> Twist {
        // Compute dynamic window (reachable velocities given acceleration limits)
        let window = self.compute_dynamic_window(current_vel);

        let mut best_trajectory: Option<Trajectory> = None;
        let mut best_cost = f64::INFINITY;

        // Sample velocities within dynamic window
        for linear_vel in self.sample_range(window.min_linear, window.max_linear, self.linear_vel_samples) {
            for angular_vel in self.sample_range(window.min_angular, window.max_angular, self.angular_vel_samples) {
                let vel = Twist { linear: linear_vel, angular: angular_vel };

                // Simulate trajectory
                let trajectory = self.simulate_trajectory(current_pose, vel);

                // Check for collisions
                if self.check_collision(&trajectory, costmap) {
                    continue;
                }

                // Compute cost
                let cost = self.compute_trajectory_cost(&trajectory, global_path, costmap);

                if cost < best_cost {
                    best_cost = cost;
                    best_trajectory = Some(trajectory);
                }
            }
        }

        best_trajectory
            .map(|t| t.velocities[0])
            .unwrap_or(Twist { linear: 0.0, angular: 0.0 })  // Stop if no valid trajectory
    }

    fn compute_dynamic_window(&self, current_vel: Twist) -> DynamicWindow {
        let dt = self.sim_time;

        DynamicWindow {
            min_linear: (current_vel.linear - self.max_linear_acc * dt).max(self.min_linear_vel),
            max_linear: (current_vel.linear + self.max_linear_acc * dt).min(self.max_linear_vel),
            min_angular: (current_vel.angular - self.max_angular_acc * dt).max(-self.max_angular_vel),
            max_angular: (current_vel.angular + self.max_angular_acc * dt).min(self.max_angular_vel),
        }
    }

    fn simulate_trajectory(&self, start: Isometry2<f64>, vel: Twist) -> Trajectory {
        let mut poses = vec![start];
        let mut velocities = vec![vel];
        let mut current = start;

        let steps = (self.sim_time / self.sim_granularity) as usize;

        for _ in 0..steps {
            let theta = current.rotation.angle();
            let dx = vel.linear * self.sim_granularity * theta.cos();
            let dy = vel.linear * self.sim_granularity * theta.sin();
            let dtheta = vel.angular * self.sim_granularity;

            current = Isometry2::new(
                current.translation.vector + Vector2::new(dx, dy),
                theta + dtheta,
            );

            poses.push(current);
            velocities.push(vel);
        }

        Trajectory { poses, velocities, time_step: self.sim_granularity }
    }

    fn compute_trajectory_cost(
        &self,
        trajectory: &Trajectory,
        global_path: &[Isometry2<f64>],
        costmap: &Costmap,
    ) -> f64 {
        let endpoint = trajectory.poses.last().unwrap();

        // Distance to path
        let path_dist = self.distance_to_path(endpoint, global_path);

        // Distance to goal (final path point)
        let goal_dist = if let Some(goal) = global_path.last() {
            (endpoint.translation.vector - goal.translation.vector).norm()
        } else {
            0.0
        };

        // Obstacle cost (integrated along trajectory)
        let obstacle_cost: f64 = trajectory.poses.iter()
            .map(|p| costmap.cost_at(p.translation.vector))
            .sum::<f64>() / trajectory.poses.len() as f64;

        self.path_distance_weight * path_dist
            + self.goal_distance_weight * goal_dist
            + self.obstacle_weight * obstacle_cost
    }
}
```

```rust
/// Trajectory tracking controller using feedback linearization
pub struct TrajectoryTracker {
    /// Control gains
    k_rho: f64,    // Distance gain
    k_alpha: f64,  // Heading to goal gain
    k_beta: f64,   // Final heading gain
    /// Limits
    max_linear_vel: f64,
    max_angular_vel: f64,
}

impl TrajectoryTracker {
    /// Compute velocity command to track a target pose
    pub fn compute(&self, current: Isometry2<f64>, target: Isometry2<f64>) -> Twist {
        // Error in robot frame
        let dx = target.translation.x - current.translation.x;
        let dy = target.translation.y - current.translation.y;
        let current_theta = current.rotation.angle();
        let target_theta = target.rotation.angle();

        // Transform to polar coordinates
        let rho = (dx * dx + dy * dy).sqrt();
        let alpha = normalize_angle(dy.atan2(dx) - current_theta);
        let beta = normalize_angle(target_theta - current_theta - alpha);

        // Control law (smooth, stable)
        let v = self.k_rho * rho;
        let omega = self.k_alpha * alpha + self.k_beta * beta;

        // Handle backward motion for large heading errors
        let (v, omega) = if alpha.abs() > std::f64::consts::FRAC_PI_2 {
            (-v, -omega)
        } else {
            (v, omega)
        };

        Twist {
            linear: v.clamp(-self.max_linear_vel, self.max_linear_vel),
            angular: omega.clamp(-self.max_angular_vel, self.max_angular_vel),
        }
    }

    /// Track a trajectory (call at high rate, e.g., 100Hz)
    pub fn track_trajectory(
        &self,
        current: Isometry2<f64>,
        trajectory: &Trajectory,
        time_since_start: f64,
    ) -> Twist {
        // Find the target pose on trajectory
        let index = ((time_since_start / trajectory.time_step) as usize)
            .min(trajectory.poses.len() - 1);

        // Look ahead slightly for smoother tracking
        let lookahead_index = (index + 2).min(trajectory.poses.len() - 1);
        let target = trajectory.poses[lookahead_index];

        self.compute(current, target)
    }
}
```

```rust
/// Costmap layer
pub trait CostmapLayer: Send + Sync {

    fn get_cost(&self, x: usize, y: usize) -> u8;
}

/// Static map layer (from SLAM)
pub struct StaticLayer {
    map: OccupancyGrid,
}

/// Obstacle layer (from recent LiDAR)
pub struct ObstacleLayer {
    data: Array2<u8>,
    /// Observation buffer for temporal filtering
    observations: VecDeque<(Instant, Vec<(usize, usize)>)>,
    /// How long to keep observations
    observation_persistence: Duration,
}

impl CostmapLayer for ObstacleLayer {
    fn update(&mut self, scan: &Scan, pose: &Isometry2<f64>) {
        // Mark observed cells
        let mut cells = Vec::new();
        for point in scan.points_in_frame(pose) {
            if let Some((x, y)) = self.world_to_grid(point) {
                cells.push((x, y));
                self.data[(y, x)] = LETHAL_COST;
            }
        }

        self.observations.push_back((Instant::now(), cells));

        // Clear old observations
        while let Some((time, _)) = self.observations.front() {
            if time.elapsed() > self.observation_persistence {
                if let Some((_, old_cells)) = self.observations.pop_front() {
                    for (x, y) in old_cells {
                        self.data[(y, x)] = FREE_COST;
                    }
                }
            } else {
                break;
            }
        }
    }
}

/// Inflation layer (expands obstacles by robot radius)
pub struct InflationLayer {
    data: Array2<u8>,
    inflation_radius: f64,
    inscribed_radius: f64,
    cost_scaling_factor: f64,
    resolution: f64,
}

impl InflationLayer {
    pub fn compute(&mut self, base_costmap: &Array2<u8>) {
        let inflation_cells = (self.inflation_radius / self.resolution) as i32;
        let inscribed_cells = (self.inscribed_radius / self.resolution) as i32;

        self.data.fill(FREE_COST);

        // Find all lethal cells and inflate
        for y in 0..base_costmap.nrows() {
            for x in 0..base_costmap.ncols() {
                if base_costmap[(y, x)] >= LETHAL_COST {
                    self.inflate_cell(x as i32, y as i32, inflation_cells, inscribed_cells);
                }
            }
        }
    }

    fn inflate_cell(&mut self, cx: i32, cy: i32, inflation: i32, inscribed: i32) {
        for dy in -inflation..=inflation {
            for dx in -inflation..=inflation {
                let x = cx + dx;
                let y = cy + dy;

                if x < 0 || y < 0 || x >= self.data.ncols() as i32 || y >= self.data.nrows() as i32 {
                    continue;
                }

                let dist_sq = dx * dx + dy * dy;
                let dist = (dist_sq as f64).sqrt();

                let cost = if dist_sq == 0 {
                    LETHAL_COST
                } else if dist <= inscribed as f64 {
                    INSCRIBED_COST  // Definitely in collision
                } else if dist <= inflation as f64 {
                    // Exponential decay
                    let factor = (-self.cost_scaling_factor * (dist - inscribed as f64)).exp();
                    (INSCRIBED_COST as f64 * factor) as u8
                } else {
                    continue;
                };

                let ux = x as usize;
                let uy = y as usize;
                self.data[(uy, ux)] = self.data[(uy, ux)].max(cost);
            }
        }
    }
}

/// Layered costmap
pub struct Costmap {
    layers: Vec<Box<dyn CostmapLayer>>,
    combined: Array2<u8>,
    resolution: f64,
    origin: Vector2<f64>,
}

impl Costmap {
    pub fn update(&mut self, scan: &Scan, pose: &Isometry2<f64>) {
        // Update each layer
        for layer in &mut self.layers {
            layer.update(scan, pose);
        }

        // Combine layers (max of all)
        self.combined.fill(FREE_COST);
        for layer in &self.layers {
            for y in 0..self.combined.nrows() {
                for x in 0..self.combined.ncols() {
                    let cost = layer.get_cost(x, y);
                    self.combined[(y, x)] = self.combined[(y, x)].max(cost);
                }
            }
        }
    }

    pub fn cost_at(&self, point: Vector2<f64>) -> f64 {
        let (x, y) = self.world_to_grid(point);
        if x < self.combined.ncols() && y < self.combined.nrows() {
            self.combined[(y, x)] as f64
        } else {
            LETHAL_COST as f64
        }
    }
}

const FREE_COST: u8 = 0;
const INSCRIBED_COST: u8 = 253;
const LETHAL_COST: u8 = 254;
```

```rust
/// Mission state machine
pub enum MissionState {
    Idle,

    AtWaypoint { waypoint_index: usize, arrival_time: Instant },
    Paused,
    Completed,

/// A patrol mission
pub struct PatrolMission {
    waypoints: Vec<Isometry2<f64>>,
    loop_forever: bool,
    current_index: usize,
    state: MissionState,
    /// How long to wait at each waypoint
    dwell_time: Duration,
}

impl PatrolMission {
    pub fn new(waypoints: Vec<Isometry2<f64>>, loop_forever: bool) -> Self {
        Self {
            waypoints,
            loop_forever,
            current_index: 0,
            state: MissionState::Idle,
            dwell_time: Duration::from_secs(1),
        }
    }

    pub fn start(&mut self) {
        if !self.waypoints.is_empty() {
            self.state = MissionState::NavigatingTo { waypoint_index: 0 };
        }
    }

    pub fn update(&mut self, current_pose: Isometry2<f64>, navigation_status: NavigationStatus) {
        match &self.state {
            MissionState::Idle => {}

            MissionState::NavigatingTo { waypoint_index } => {
                match navigation_status {
                    NavigationStatus::Succeeded => {
                        self.state = MissionState::AtWaypoint {
                            waypoint_index: *waypoint_index,
                            arrival_time: Instant::now(),
                        };
                    }
                    NavigationStatus::Failed(reason) => {
                        self.state = MissionState::Failed { reason };
                    }
                    NavigationStatus::Active => {}
                }
            }

            MissionState::AtWaypoint { waypoint_index, arrival_time } => {
                if arrival_time.elapsed() >= self.dwell_time {
                    // Move to next waypoint
                    let next = waypoint_index + 1;

                    if next >= self.waypoints.len() {
                        if self.loop_forever {
                            self.current_index = 0;
                            self.state = MissionState::NavigatingTo { waypoint_index: 0 };
                        } else {
                            self.state = MissionState::Completed;
                        }
                    } else {
                        self.current_index = next;
                        self.state = MissionState::NavigatingTo { waypoint_index: next };
                    }
                }
            }

            MissionState::Paused | MissionState::Completed | MissionState::Failed { .. } => {}
        }
    }

    pub fn current_goal(&self) -> Option<Isometry2<f64>> {
        match &self.state {
            MissionState::NavigatingTo { waypoint_index } => {
                self.waypoints.get(*waypoint_index).copied()
            }
            _ => None,
        }
    }
}
```

## Integration: The Autonomy Orchestrator

Ties everything together:

```rust
/// Main autonomy system
pub struct AutonomySystem {
    // State estimation
    odometry: DifferentialOdometry,
    ekf: PoseEkf,
    slam: PoseGraph,

    // Mapping
    occupancy_grid: OccupancyGrid,
    costmap: Costmap,

    // Planning
    global_planner: HybridAStar,
    local_planner: DwaPlanner,
    trajectory_tracker: TrajectoryTracker,

    // Mission
    mission: Option<PatrolMission>,

    // Current state
    current_pose: Isometry2<f64>,
    current_velocity: Twist,
    global_path: Vec<Isometry2<f64>>,
    local_trajectory: Option<Trajectory>,
}

impl AutonomySystem {
    /// Main update loop (call at 100Hz)
    pub fn update(
        &mut self,
        scan: &Scan,
        left_vel: f64,
        right_vel: f64,
        dt: f64,
    ) -> Twist {
        // 1. State estimation
        let odom_pose = self.odometry.update(left_vel, right_vel, dt);

        // 2. Scan matching for correction
        let match_result = self.slam.scan_matcher.match_scan(
            scan,
            &self.occupancy_grid,
            odom_pose,
        );

        // 3. Fuse in EKF
        self.ekf.predict(&self.odometry.delta(), &self.odometry.covariance);
        self.ekf.update(&match_result.pose_vector(), &match_result.covariance);
        self.current_pose = self.ekf.pose();

        // 4. Update SLAM (less frequently)
        self.slam.add_scan(scan.clone(), self.current_pose);

        // 5. Update costmap
        self.costmap.update(scan, &self.current_pose);

        // 6. Mission update
        if let Some(mission) = &mut self.mission {
            mission.update(self.current_pose, self.navigation_status());

            // Check if we need a new global plan
            if let Some(goal) = mission.current_goal() {
                if self.needs_replan(&goal) {
                    self.global_path = self.global_planner
                        .plan(self.current_pose, goal, &self.costmap)
                        .unwrap_or_default();
                }
            }
        }

        // 7. Local planning (10Hz, so skip some cycles)
        if self.should_run_local_planner() {
            self.local_trajectory = Some(self.local_planner.compute_trajectory(
                self.current_pose,
                self.current_velocity,
                &self.global_path,
                &self.costmap,
            ));
        }

        // 8. Trajectory tracking (100Hz)
        let cmd = if let Some(ref trajectory) = self.local_trajectory {
            self.trajectory_tracker.track_trajectory(
                self.current_pose,
                trajectory,
                self.time_on_trajectory(),
            )
        } else {
            Twist { linear: 0.0, angular: 0.0 }
        };

        self.current_velocity = cmd;
        cmd
    }
}
```

## Booth Atmosphere: The Intangibles

### Sound

The warehouse will be loud. Don't fight it.

- No audio on video reel (competing noise is worse than silence)
- Rover motors have a satisfying hum (let it be heard)
- Your voice is the only audio that matters

### Smell

Weird but real: booths that smell bad lose visitors.

- No food at booth (eat elsewhere)
- Bring breath mints
- Electronics smell is fine

### Touch

Give people reasons to touch things:

- Xbox controller (obviously)
- Snow auger on display (let them pick it up)
- One-pagers (nice paper stock matters)
- The rover itself (when stopped, invite them to feel the build quality)

### Movement

The rover moving is your unfair advantage.

- Never let it sit idle for more than 30 seconds
- If no one is actively demoing, it should be patrolling
- Movement draws eyes from across the room

### Height

Monitors should be at eye level or slightly above.

- Standing visitors: monitors at 5'6" center
- Rover on ground is fine (people look down at robots)
- Banner above everything (visible from 30 feet)

## Investor Qualification

Not every conversation is worth 10 minutes. Qualify quickly.

### Green Flags (Invest Time)

- "We invest in hardware/robotics"
- "I've looked at this space before"
- "What's your timeline for the raise?"
- "Can you send me materials?"
- Asks specific questions about unit economics
- Takes notes or photos

### Yellow Flags (Keep Brief)

- "Interesting, but we don't do hardware"
- "We only invest at Series A+"
- General curiosity, no investment intent
- Competitor research

### Red Flags (Polite Exit)

- Wants to pitch you their startup
- Just wants to drive the robot for fun
- Asks about job openings (fine, but not now)
- Visibly not paying attention

### Exit Lines

- "Great meeting you, grab a one-pager!"
- "Let me let you explore the demo. Come back if you have questions."
- "I should check on the rover. Nice to meet you."

## Competition at Artifact

You'll share the floor with other startups. Stand out.

### What Others Will Have

- Laptop and poster
- Maybe a demo video
- Founder standing awkwardly

### What You Have That They Don't

- A physical robot that moves
- Interactive demo (they can participate)
- Professional booth setup
- Real hardware, not mockups
- Cleveland grit

### Positioning

- You're not a student project
- You're not a consumer gadget
- You're building critical infrastructure
- You ship in the Midwest, not SF

## Mental Preparation

### Before Festival

- Get sleep the night before
- Eat a real breakfast
- Arrive early, don't rush setup
- Run demo 3x before doors open
- Breathe

### During Festival

- Stay hydrated
- Eat during lulls
- Take 5 min breaks every 2 hours
- Don't check email/Slack (focus on conversations)
- It's a performance: stay energized

### When It Goes Wrong

- Hardware will glitch. It's fine.
- Have your fallback ready.
- Laugh it off: "Classic demo gremlins. Let me show you on video."
- The pitch matters more than perfect demos.
